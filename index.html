üåê AWS Services & Integration Flow

Below is an overview of the core AWS services that power Neural Climate and how data flows through the system:

  • IoT Sensors collect environmental metrics like temperature, humidity, and vibration.
  • AWS IoT Core securely receives sensor data using MQTT.
  • Lambda Functions filter and preprocess data, generate alerts.
  • Amazon Timestream stores telemetry for historical analysis.
  • Amazon S3 stores raw logs, model training data, media.
  • Amazon SageMaker detects early warning signs of faults.
  • Amazon QuickSight displays dashboards in real time.
  • AWS Step Functions automate safety and control workflows.
